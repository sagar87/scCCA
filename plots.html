scCCA.plots.qc.disp(adata, model_key='scpca', layers_key=None, protein_obsm_key=None, cmap='viridis', ax=None)

Plot the fitted dispersion against the coefficient of variation RNA or protein counts.

Parameters:

  • adata (AnnData) – AnnData object.

  • model_key (str, optional (default: "scpca")) – Key for the fitted model.

  • layers_key (str, optional (default: None)) – If layers_key is None, then the raw counts are extracted from adata.X. Otherwise, the counts are extracted from adata.layers[layers_key].

  • protein_obsm_key (str, optional (default: None)) – Key for protein counts in adata.obsm. Providing protein_obsm_key overrides layers_key, i.e. protein counts are plotted.

  • cmap (str, optional (default: "viridis")) – Colormap for the scatterplot. Color represents the mean of the counts.

  • ax (matplotlib.axes.Axes, optional (default: None)) – Axes to plot on. If None, then a new figure is created.

Returns:

ax

Return type:

matplotlib.axes.Axes

Visualize the loading state of a given model on the data.

Parameters:

  • adata (AnnData) – The annotated data matrix.

  • model_key (str) – Key to access the model from the data.

  • states (Union[str, List[str]]) – List of states to consider. Default is an empty list.

  • factor (Union[int, List[int], None]) – Factor or factors to consider. Default is None.

  • variable (str) – Variable to consider. Default is “W_rna”.

  • highest (int) – Number of highest values to consider. Default is 10.

  • lowest (int) – Number of lowest values to consider. Default is 10.

  • threshold (Optional[float]) – Threshold value to consider. Default is None.

  • sign (int) – Sign value. Default is 1.0.

  • geneset (Optional[str]) – Set of genes to consider. Default is None.

  • geneset_top_genes (int) – Number of top genes in the gene set to consider. Default is 100.

  • geneset_bottom_genes (int) – Number of bottom genes in the gene set to consider. Default is 0.

  • organism (str) – Organism to consider. Default is “Human”.

  • cmap (Colormap) – Colormap to use. Default is cm.RdBu.

  • colorbar_pos (str) – Position of the colorbar. Default is “right”.

  • colorbar_width (str) – Width of the colorbar. Default is “3%”.

  • orientation (str) – Orientation of the plot. Default is “vertical”.

  • fontsize (int) – Font size to use. Default is 10.

  • pad (float) – Padding value. Default is 0.1.

  • show_corr (bool) – Whether to show correlation. Default is False.

  • show_rank (bool) – Whether to show rank. Default is False.

  • show_diff (bool) – Whether to show difference. Default is False.

  • show_lines (bool) – Whether to show lines. Default is False.

  • size_func (Callable) – Function to determine size. Default is a function that returns 10.

  • text_func (Callable) – Function to format text. Default is a function that wraps text.

  • sharey (bool) – Whether to share y-axis. Default is False.

  • sharex (bool) – Whether to share x-axis. Default is False.

  • ncols (int) – Number of columns. Default is 4.

  • width (int) – Width of the plot. Default is 4.

  • height (int) – Height of the plot. Default is 3.

  • text_kwargs (dict) – Keyword arguments for text formatting. Default is an empty dictionary.

  • ax (Optional[Axes]) – Axis object. Default is None.

Returns:

Configured axis object.

Return type:

ax

Notes

This function visualizes the loading state of a given model on the data. It provides various customization options for visualization.

scCCA.plots.cluster_enrichment.cluster_enrichment(adata, model_key, states, factor, state_key, cluster_key, groups, dot=False, sign=1, highest=0, lowest=0, threshold=1.96, return_var_names=False, **kwargs)

Enrichment analysis of clusters based on differential gene expression.

Parameters:

  • adata (AnnData) – Anndata object.

  • model_key (str) – Key to access the model in the adata object.

  • states (Union[str, List[str]]) – States for comparison. If not pair of states is provided, “Intercept” is assumed to be the base state.

  • factor (int) – Factor index to consider for the differential calculation.

  • state_key (str) – Key for state in adata.

  • cluster_key (str) – Key for cluster in adata.

  • groups (Union[str, List[str]]) – Groups to consider for enrichment.

  • dot (bool) – If True, a dot plot is generated. Otherwise, a heatmap is generated. Default is False.

  • sign (Union[int, float]) – Sign to adjust the difference, by default 1.

  • highest (int) – Number of highest differential genes to retrieve, by default 0.

  • lowest (int) – Number of lowest differential genes to retrieve, by default 0.

  • threshold (float) – Threshold for significance, by default 1.96.

  • return_var_names (bool) – If True, returns variable names. Otherwise, returns the plot axes. Default is False.

  • **kwargs – Additional keyword arguments passed to the plotting function.

Returns:

If return_var_names is True, returns a dictionary of variable names. Otherwise, returns the plot axes.

Return type:

Union[dict, matplotlib.axes.Axes]

Notes

This function performs enrichment analysis of clusters based on differential gene expression. It first validates the sign and retrieves differential genes. Depending on the dot parameter, either a dot plot or a heatmap is generated to visualize the enrichment.
